# SQL_job_market_analysis
PostgreSQL project analyzing a job postings dataset to identify hiring trends, role demand, skill demand and and time-based market patterns using advanced SQL queries.

**Tech Stack:** PostgreSQL, SQL  
**Dataset:** Job Postings Dataset ( From : https://www.lukebarousse.com/ , https://www.youtube.com/watch?v=7mz73uXD9DA )

Project is an extension of/inspired by https://www.youtube.com/watch?v=7mz73uXD9DA

### questions answered

What are the top-paying data analyst jobs?
What skills are required for these top-paying jobs?
What skills are most in demand for data analysts?
Which skills are associated with higher salaries?
What are the most optimal skills to learn?
